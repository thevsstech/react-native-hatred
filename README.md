# react-native-hatred

React Native UI Library will help you to reduce your hatred towards react native

## Installation

```sh
npm install react-native-hatred
```

## Usage

```js
import Hatred from "react-native-hatred";

// ...

const result = await Hatred.multiply(3, 7);
```
